# cezoom-msa-assets

### Description
This is the main service for user related resources. It shares the MongoDB integration.

### Enviroment
- Required enviroment variables located on `envs.example`
- `JWT_SECRET` needs to be the same across all msa projects
- `USER_URL` refers to the cezoom-msa-users resource

### How to run
`npm start`

# API
